# Bitonic-Sorting-
C++ implementation of Bitonic sorting and searching. The project splits an array into two halves: the first is sorted ascendingly using Bubble Sort, and the second descendingly using Selection Sort, followed by a search operation.
# Bitonic Sort and Search (C++ Project)


This project demonstrates a practical application of sorting and searching algorithms using C++ as part of my Data Structures studies.

## Project Description
The program processes an array to create a bitonic sequence (increasing then decreasing) by:
1. Sorting the first half of the array **ascendingly** using the **Bubble Sort** algorithm.
2. Sorting the second half of the array **descendingly** using the **Selection Sort** algorithm.
3. Performing a **Linear Search** to find a specific target element within the modified array.

## Features
* **Sorting Algorithms:** Implementation of Bubble Sort and Selection Sort.
* **Performance Metrics:** Tracks the number of comparisons and swaps performed during sorting.
* **Test Cases:** Includes various scenarios such as searching for the peak element, searching in both halves, handling duplicate values, and handling missing elements.
